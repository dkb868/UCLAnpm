#Dining Hall
Usage:
```
var options = {
	diningHall: 0, //0-3, BPlate, Covel, De Neve, Feast
	meal: 2, //0-2, breakfast, lunch, dinner
	diet: 0, //0-2, normal, vegetarian, vegan
	callback: printMenu //callback with 1 param
};
```

Then call:

```
UCLA.getMenu(options);
```
